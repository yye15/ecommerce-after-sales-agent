# 问题分类体系

本项目的问题分类不只判断评论是正面还是负面，而是把售后文本转成可执行的业务归因。分类结果保留兼容字段 `category`，同时增加业务维度和二级原因，方便客服、售后和运营分别使用。

## 输出结构

每个分类结果包含：

```text
category             一级问题类别
primary_category     与 category 保持一致，便于后续扩展
business_dimension   业务维度
secondary_reasons    二级原因
confidence           规则置信度
evidence             命中文本证据
negative_count       负面证据数量
matched_keywords     命中的规则关键词
```

## 一级类别

| 一级类别 | 业务维度 | 说明 |
|---|---|---|
| 商品质量问题 | 商品 | 破损、故障、耐用性、真假、新鲜度等问题 |
| 产品体验问题 | 体验 | 使用体验、功能感知、适配、气味、噪音等问题 |
| 物流问题 | 履约 | 发货、运输、配送、签收等问题 |
| 客服响应问题 | 服务 | 客服响应、售后处理、服务态度等问题 |
| 价格与权益问题 | 权益 | 价格、优惠、发票、赠品、宣传承诺等问题 |
| 退换货与赔付问题 | 售后 | 退款、退货、换货、保修、补偿等诉求 |
| 其他反馈 | 其他 | 暂未命中明确售后问题类别 |

## 二级原因示例

| 一级类别 | 二级原因示例 |
|---|---|
| 商品质量问题 | 破损/瑕疵、核心故障、耐用性不足、材质/做工问题、真假/描述可信度、食品新鲜度 |
| 产品体验问题 | 核心功能体验、性能/稳定性、适配/安装、感官体验、内容/图片不符 |
| 物流问题 | 发货延迟、配送异常、履约沟通受阻 |
| 客服响应问题 | 响应慢或失联、处理拖延、服务态度差 |
| 价格与权益问题 | 价格争议、优惠/赠品争议、票据/运费、描述不符/误导 |
| 退换货与赔付问题 | 退款诉求、退货/换货诉求、补偿/赔付诉求、保修/维修 |
